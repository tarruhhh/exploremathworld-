High School Math

Content about high school-level math will go here!

This guide takes you on a comprehensive journey through high school math, covering essential concepts and skills with a focus on clarity, examples, and problem-solving. We'll explore algebra, geometry, trigonometry, and pre-calculus, providing numerous formulas, diverse problems, and detailed explanations to solidify your understanding.

  1. Algebra

    • Variables and Expressions
      •
      Variables: Letters (like x,y,z) representing unknown values.
      - Example: In the expression 2x + 3y, 'x' and 'y' are variables representing unknown numbers.
      Expressions: Combinations of variables, numbers, and operations (+, -, ×, ÷).
      - Example: 3x + 2y - 5 is an algebraic expression with variables x and y.
      - Example: 4a² - 7b + 1 is another expression, combining variables 'a' and 'b' with exponents and constants.

    • Equations and Inequalities

      • Equations: Statements of equality between two expressions.
        • - Formula: expression 1 = expression 2
        - Example: 2x + 5 = 11
        - Explanation: This equation states that the expression "2x + 5" is equal to the value "11". The goal is to find the value of 'x' that makes this true.
      • Inequalities: Statements of comparison between two expressions using symbols: <, >, ≤, ≥.
        • - Formula: expression 1 < expression 2 (or >, ≤, ≥)
        - Example: x - 3 > 2
        - Explanation: This inequality states that the expression "x - 3" is greater than the value "2". The goal is to find all values of 'x' that satisfy this condition.

    • Solving Equations and Inequalities

      • Solving Equations: Finding the values of variables that make the equation true.

        • - Methods:
        • Substitution:
          • Replacing a variable with its known value.
        • - Example:
          • If we know y = 3, we can substitute '3' for 'y' in the equation 2x + 3y = 10 to get 2x + 3(3) = 10.
        • Elimination:
          • Combining equations to eliminate a variable.
        • - Example:
          • To solve the system 2x + y = 5 and x - y = 1, we can add the equations together to eliminate 'y', resulting in 3x = 6, and then solve for 'x'.
        • Factoring:
          • Finding expressions that multiply to give the original expression.
        • - Example:
          • To solve x² - 4x + 3 = 0, we factor the quadratic expression into (x - 1)(x - 3) = 0.
        • Quadratic Formula:
          • Solving for x in equations of the form ax² + bx + c = 0.
        • Formula (Quadratic Formula):
          • x = (-b ± √(b² - 4ac)) / 2a
        • - Example:
          • To solve the equation 2x² - 5x + 3 = 0, we use the quadratic formula: x = (5 ± √((-5)² - 4 * 2 * 3)) / (2 * 2) = (5 ± √1) / 4. This gives us two solutions: x = 1 and x = 3/2.

    • Solving Inequalities: Finding the values of variables that make the inequality true.
      • Similar methods to solving equations, but with additional rules for inequality signs:
        • - Multiplying or dividing by a negative number flips the inequality sign.
      • Example:
      1. Add 3 to both sides: x > 5
      2. The solution is all values of 'x' greater than 5.

    • Functions

      • - Functions: Relationships between input values (domain) and output values (range). Often expressed as equations or graphs.
      • Formula:
        • f(x) = expression (where x is the input and f(x) is the output)
      • - Example:
        • f(x) = 2x + 1 is a function where the input x is multiplied by 2 and then 1 is added to get the output f(x).
      • - Example:
        • If we input x = 3 into the function f(x) = 2x + 1, the output is f(3) = 2(3) + 1 = 7.

    • Linear Equations and Systems of Equations

      • - Linear Equations: Represent straight lines.
      • Formula (Slope-intercept form):
        • y = mx + c (m is the slope, c is the y-intercept)
      • - Example:
        • y = 2x + 3 is a linear equation with slope 2 and y-intercept 3.
      • Explanation:
        • This equation tells us that for every increase of 1 in the 'x' value, the 'y' value increases by 2. The y-intercept 'c' tells us where the line crosses the y-axis.

      - Systems of Equations: Two or more equations with multiple variables.
      • Example: The system of equations:
        • 2x + y = 5, x - y = 1
      • Solving Methods:

      • Substitution: Solving one equation for one variable and substituting it into the other equation.
        • - Example: Solving the second equation (x - y = 1) for 'x', we get x = y + 1. Substituting this into the first equation (2x + y = 5), we get 2(y + 1) + y = 5. Solving for 'y', we find y = 1. Substituting this back into x = y + 1, we find x = 2.
      • Elimination: Adding or subtracting equations to eliminate a variable.
        • - Example: We can add the two equations together to eliminate 'y': (2x + y) + (x - y) = 5 + 1. This simplifies to 3x = 6, and solving for 'x', we get x = 2. Substituting this back into either original equation, we find y = 1.

    • Quadratic Equations

      • - Quadratic Equations: Equations with a highest power of 2 for the variable.
      • Formula (Standard Form): ax² + bx + c = 0
        • - Example: x² - 4x + 3 = 0

      - Solving Methods:
      • Factoring: Finding expressions that multiply to give the original expression.
        • - Example: x² - 4x + 3 = 0. To solve x² - 4x + 3 = 0, we factor the quadratic expression into (x - 1)(x - 3) = 0. This means either (x - 1) = 0 or (x - 3) = 0, giving solutions x = 1 and x = 3.
      • Completing the Square: Manipulating the equation to create a perfect square trinomial.
        • - Example: To solve x² - 6x + 5 = 0 by completing the square:
      1. Move the constant term to the right side: x² - 6x = -5
      2. Take half of the coefficient of
      3. Factor the left side as a perfect square: (x - 3)² = 4
      4. Take the square root of both sides: x - 3 = ±2
      5. Solve for 'x': x = 3 ± 2, giving solutions x = 5 and x = 1.

      - Quadratic Formula: Solving for x in equations of the form ax² + bx + c = 0.
      • Formula (Quadratic Formula): x = (-b ± √(b² - 4ac)) / 2a
        • - Example: To solve the equation 2x² - 5x + 3 = 0, we use the quadratic formula: x = (5 ± √((-5)² - 4 * 2 * 3)) / (2 * 2) = (5 ± √1) / 4. This gives us two solutions: x = 1 and x = 3/2.

    • Polynomial Equations

      • - Polynomial Equations: Equations involving expressions with multiple terms, each containing a variable raised to a non-negative integer power.
      • Formula (General form):
        • aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀ = 0 (where n is a non-negative integer)
        - Example: x³ + 2x² - 5x + 1 = 0 is a polynomial equation.

    • Exponents and Radicals

        • - Exponents: Indicate repeated multiplication.
      • Formula:
        • xⁿ = x * x * x * ... * x (n times)
        - Example: x³ = x * x * x
        - Radicals: Represent the opposite operation of exponents (like square roots).
      • Formula:
        • √a = b (where b² = a)
        - Example: √9 = 3 (because 3² = 9)

    • Logarithms

        • - Logarithms: The inverse of exponents. They help solve equations where the variable is in the exponent.
      • Formula:
        • logₐb = c (where aᶜ = b)
        - Example: log₂8 = 3 (because 2³ = 8)
